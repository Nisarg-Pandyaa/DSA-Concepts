# ASCII Values
- 'a' : 97,...,'z' : 122
- 'A' : 65,...,'Z' : 90
- 'a' - 'A' = 32

----
- `String.valueOf(...)` :
  - `variable` ki actual as it is value ko string me convert kar dega. 45 -> "45", flag = true -> "true", Object obj = null; => "null", etc.
  - `literals` ko as it is string me convert kar dega.
  - `char arr` ko string me convert kar dega.

----
- `for(int i=0;i<n;i++)` is faster than `for(int num : arr)`
